# **App Name**: OrganiZen

## Core Features:

- Annual Calendar: Display an annual calendar view where the user can see all months of the year at once.
- Add Annual Events: Allow users to add and view important dates/events on the annual calendar.
- Daily Schedule: Provide a daily schedule view where users can input their morning and afternoon activities/classes by time.
- Monthly Calendar: Enable users to set up a monthly calendar for planning and noting specific tasks, appointments, and commitments.
- Weekly Calendar: Display a weekly calendar providing space between days for the user to add daily activities, to-dos, or short reminders.
- Smart Event Reminder: Smart event reminder: using natural language processing, analyze userâ€™s added task and, with help of an LLM, tool set a smart reminder.

## Style Guidelines:

- Primary color: Soft Pink (#F4C2C2) to evoke a sense of calm and organization.
- Background color: Light Beige (#F5F5DC) as a neutral backdrop to ensure readability and a sense of spaciousness.
- Accent color: Muted Lavender (#D8BFD8) to highlight key interactive elements, maintaining a consistent user experience.
- Body and headline font: 'PT Sans', a humanist sans-serif suitable for both headlines and body text to create a clear and friendly reading experience.
- Simple, outline-style icons in Lavender to represent different event types (e.g., work, meals, appointments).
- Use a clean, minimalist layout with generous spacing to avoid clutter and improve usability.
